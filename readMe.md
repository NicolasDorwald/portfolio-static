# Portfolio Personnel

Ce projet est mon **portfolio personnel**, conçu comme une **Single Page Application (SPA) simple et sobre** pour présenter mes projets, mes compétences et permettre de me contacter facilement.

---

## Technologies utilisées

- **HTML5 & CSS3** – structure et mise en forme des pages, animations CSS pour le design.  
- **JavaScript** – interactions, animations et navigation entre les sections de la SPA.  
- **API Formspree** – gestion du formulaire de contact, envoi des messages sans backend.  

---

## Fonctionnalités

- **Navigation fluide** via des boutons qui servent de barre de navigation.  
- **Section “Projets”** : présentation de quelques projets réalisés.  
- **Section “Compétences”** : résumé de mes compétences techniques et outils maîtrisés.  
- **Formulaire de contact** : envoi direct via l’API Formspree avec confirmation d’envoi.  
- **Parcours professionnel** : bouton dédié pour consulter mon expérience.  
- **Animations** : effets CSS et JS pour améliorer l’expérience utilisateur et le design.

Ce portfolio est conçu pour être **léger, facile à maintenir et entièrement navigable depuis une seule page**, tout en mettant en avant mes projets et compétences de manière claire et accessible.
